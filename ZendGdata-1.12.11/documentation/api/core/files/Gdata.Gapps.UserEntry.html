\Zend_Gdata_Gapps_UserEntry
jump to top

Package: Zend_Gdata\Gapps
Data model class for a Google Apps User Entry.
Each user entry describes a single user within a Google Apps hosted domain. To transfer user entries to and from the Google Apps servers, including creating new entries, refer to the Google Apps service class, Zend_Gdata_Gapps. This class represents in the Google Data protocol.
