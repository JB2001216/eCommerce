\Zend_Gdata_Photos_PhotoEntry

Package: Zend_Gdata\Photos
Data model class for a Comment Entry.
To transfer user entries to and from the servers, including creating new entries, refer to the service class, Zend_Gdata_Photos. This class represents in the Google Data protocol.
